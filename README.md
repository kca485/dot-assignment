# DOT Assignment

## Instalasi

1. Klon repositori ini

```bash
git clone https://github.com/kca485/dot-assignment.git
```

2. Masuk ke direktori

```bash
cd dot-assignment
```

3. Instal

```bash
npm ci
```

4. Sediakan berkas .env

```bash
cp .env.example .env
```

5. Jalankan peladen lokal

```bash
npm run dev
```

6. Lihat pada URL: http://localhost:3000
